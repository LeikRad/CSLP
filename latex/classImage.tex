{\footnotesize\ttfamily void Image\+::\+Gaussian\+Filter (\begin{DoxyParamCaption}\item[{int}]{kernel\+Size,  }\item[{double}]{sigma }\end{DoxyParamCaption})}



Apply a Gaussian filter to the image. 

The {\ttfamily Gaussian\+Filter} method applies a Gaussian filter to the image using the specified kernel size and sigma value.

Gaussian\+Filter is a filter commonly used in image processing for smoothing, reducing noise, and computing derivatives of an image. It is a convolution-\/based filter that uses a Gaussian matrix as its underlying kernel. In this implementation, we first create a Gaussian kernel with the specified kernel\+Size and sigma values. We then normalize the kernel so that the sum of all values is equal to 1. We then apply the Gaussian filter to the image by convolving the kernel with each pixel in the image. We use a temporary buffer to store the original pixel values so that we can apply the filter in-\/place.
