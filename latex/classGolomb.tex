\doxysubsection{Detailed Description}
A class for encoding and decoding non-\/negative integers using \doxylink{classGolomb}{Golomb} coding. 

The \doxylink{classGolomb}{Golomb} class provides methods for encoding and decoding non-\/negative integers using \doxylink{classGolomb}{Golomb} coding with a specified divisor M. The class uses a \doxylink{classBitStream}{Bit\+Stream} object to read from and write to an input/output file.
