Detailed Description

A class representing a digital image.

The Image class represents a digital image, with methods for reading and writing image files, applying various image processing operations, and displaying the image on the screen.

◆ GaussianFilter()

void Image::GaussianFilter ( int  kernelSize,
double  sigma 
)

Apply a Gaussian filter to the image.

The GaussianFilter method applies a Gaussian filter to the image using the specified kernel size and sigma value.

GaussianFilter is a filter commonly used in image processing for smoothing, reducing noise, and computing derivatives of an image. It is a convolution-based filter that uses a Gaussian matrix as its underlying kernel. In this implementation, we first create a Gaussian kernel with the specified kernelSize and sigma values. We then normalize the kernel so that the sum of all values is equal to 1. We then apply the Gaussian filter to the image by convolving the kernel with each pixel in the image. We use a temporary buffer to store the original pixel values so that we can apply the filter in-place. Note that this implementation assumes that the pixels array contains BGR color values.

Apply threshold segmentation to the image.

Image thresholding is a type of image segmentation that divides the foreground from the background in an image. In this technique, the pixel values are assigned corresponding to the provided threshold values. In computer vision, thresholding is done in grayscale images.

The ThresholdSegmentation method applies threshold segmentation to the image using the specified threshold value.
